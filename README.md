Overview

A web-based platform where users can post bugs with a bounty amount, and other users can submit solutions with proof.
The bug creator reviews submissions, selects a winner, and the bounty is awarded at a logic level.

**Key Features**
User registration and login
Create bugs with title, description, and bounty
Public listing of all bugs
Submit solutions with proof
Bug creator can approve one winner
Bug status tracking (Open / In Review / Closed)
Reward tracking for winners

**Tech Stack**
React, TypeScript, Tailwind CSS
Node.js, Express
MongoDB
